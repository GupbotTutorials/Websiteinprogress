ArduBlock

ArduBlock was created as a programming platform that simplifies physical computing with Arduino for novices. Rather than dealing with coding, fretting about syntax, and mistakenly placing semicolons, ArduBlock enables users to create programs visually using a list of code blocks that snap together.

Arduino IDE

The Arduino IDE is a software that is available as an open-source tool to write and transfer code to Arduino boards. The application can operate on various operating systems, including Windows, Mac OS X, and Linux. It facilitates the use of programming languages like C and C++.
